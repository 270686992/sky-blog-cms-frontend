## 快速上手

```sh
# 1. 安装依赖
npm install or yarn install

# 2. 修改项目根目录下的 .env.development 文件中的后端 API 为相应地址
ENV = 'development'
VUE_APP_BASE_URL = 'http://127.0.0.1:5000/'

# 3. 开发环境运行
npm run serve or yarn (run) serve
```
